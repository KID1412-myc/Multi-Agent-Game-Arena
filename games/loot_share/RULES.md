# 分赃 — 游戏规则

## 概述

6 名玩家，各自有一个公开身份和一个秘密目标。
每轮一笔随机赃款（5-15），大家各自开价想要的份额。
求和 ≤ 赃款则按出价分配，> 则谈判破裂全员空手。

## 信息公开（三阶段）

| 轮次 | 可见信息 |
|------|----------|
| 1-3 | 全盲——连自己的积分都看不到。只知道赃款数额和谈判结束。 |
| 4 | 开灯——全员积分榜公开 + 破裂次数公开 |
| 5 | 全员积分 + **上轮每人开了多少** |
| 6 | 全员积分 + **第5轮每人开了多少**（最后一轮，后果自负） |

## 公开身份（所有人可见）

- **老大**、**会计**、**打手**、**骗子**、**小偷**、**线人**
- 身份随机分配，每轮私密信息中列出谁是谁
- 身份提供思考框架，但任何身份可能有任何目标

## 秘密目标（仅自己知道）

每人随机领一个，和身份不相干：

| 目标 | 获胜条件 |
|------|----------|
| 称霸 | 总分最高（必须独赢，不能并列） |
| 垫底 | 总分全场最低 + 总分≥6（多人同最低都达成） |
| 中游 | 总分排名居中（同分同排名） |
| 毁灭者 | ≥3 回合谈判破裂 + 总分排名上半区（同分同排名） |
| 和平奖 | ≥3 回合谈判成功 + 总分严格高于全场最低分 |
| 盯上 | 总分 > 称霸目标玩家（天生对位，但不知对方是称霸） |

## 每轮流程

1. **公布赃款**：本轮赃款数额（5-15 随机）
2. **公开发言**：争论分配方案——顺序随机轮换
3. **秘密出价**：写你想要的份额

## 结算

- 所有人出价求和 ≤ 赃款 → 谈判成功，各自拿到出价的分数
- 求和 > 赃款 → 谈判破裂，全员拿 0 分。每两次破裂淘汰 1 人：同一人连续最高则定向淘汰，否则随机淘汰

## 胜利条件

6 轮后，**达成自己秘密目标的人获胜**（平则共胜）。

## 策略要点

- 你只知道自己的目标，别人的目标要从发言和出价中推理
- 公开身份只是参考——会计可能是毁灭者，老大可能是和平奖
- 垫底和毁灭者都想谈判炸——两个人在暗中互相助攻却不自知
- 过分开价会爆掉谈判，太过保守则被其他人瓜分
