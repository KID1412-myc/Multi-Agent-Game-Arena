# 狼人杀 — 游戏规则

## 概述

9 人局：3 狼人 + 预言家 + 女巫 + 猎人 + 3 平民。
白天投票淘汰，夜晚狼人刀人+神职行动。一方全灭则另一方获胜。

## 回合

| 轮次 | 阶段 |
|------|------|
| 奇数（夜晚） | 狼私聊(2轮) → 狼人选目标(≥2票) → 预言家查验 → 女巫救/毒 → 公布死亡 |
| 偶数（白天） | 公开发言 → 投票淘汰 → 遗言 → DM 公布 |

## 角色

| 角色 | 人数 | 能力 |
|------|------|------|
| 狼人 | 3 | 夜晚协商后杀一人，≥2票同一目标即成功。可刀自己骗药。私聊只狼人可见。 |
| 预言家 | 1 | 每夜查验一人身份，结果只有自己知道。白天可宣称（可真可假）。 |
| 女巫 | 1 | 解药(1次)可救活被狼杀的人，毒药(1次)可毒杀一人。每夜只能用一种。 |
| 猎人 | 1 | 被刀死或被投票出局时可开枪带走一人。被毒死不能开枪。 |
| 平民 | 3 | 无特殊能力，靠发言和投票找出狼人。 |

## 胜负

- 所有狼人死亡 → 神职+村民获胜
- 狼人数 ≥ 存活人数一半 → 狼人获胜
- 无轮次上限，直到一方达成胜利条件

## 提示

- 所有操作使用玩家ID（p1-p9）
- 死者不能发言和投票
- 猎人被投票出局时：亮身份 → 开枪（可选） → 遗言 → 淘汰
- 猎人被狼刀死时：亮身份 → 开枪（可选） → 淘汰（无遗言）
- 猎人被毒死时：直接淘汰（无开枪、无遗言）
