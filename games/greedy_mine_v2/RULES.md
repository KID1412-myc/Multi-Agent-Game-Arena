# 🛑 《贪婪矿场：零和掠夺》官方游戏规则书

## 一、基础设定
- **玩家人数：** 6 人大乱斗
- **初始资产：** 每人开局 **10 枚金币**。金币归零即刻淘汰
- **游戏时长：** 固定 **5 个回合**
- **胜利条件：** 5 回合结束后，存活且金币最多者获胜（平局则共同获胜）

## 二、核心循环

每回合两个阶段：

### 阶段一：公开发言
所有存活玩家自由发言。结盟、威胁、装可怜、承诺互不侵犯——但承诺不受系统保护，背叛被鼓励。

### 阶段二：暗箱行动
全场闭眼。每人提交行动编码（三选一）：

| 编码 | 含义 |
|------|------|
| `DEV` | 发育：挖矿 |
| `DEF` | 防御：架盾 |
| `LOOT-pX` | 掠夺：抢玩家 pX（p1~p6） |

## 三、结算矩阵

| 你的行动 | 遭遇 | 结果 |
| :--- | :--- | :--- |
| DEV | 无人掠夺你 | 金币 **+2** |
| DEV | 有人掠夺你 | 每 1 人掠夺，金币 **-4** |
| DEF | 无人掠夺你 | 金币 **-1**（护盾维持费） |
| DEF | 被 1-2 人掠夺 | 金币**不变**，掠夺者各 **-1** |
| DEF | **被 3+ 人围殴** | **破盾！防御者 -3，攻击者各 +3** |
| LOOT | 目标 DEV | 金币 **+5** |
| LOOT | 目标 DEF（未被破盾） | 金币 **-1** |
| LOOT | 目标 DEF（已被破盾） | 金币 **+3**（围殴分红） |
| LOOT | 目标也 LOOT | 双方各 **-3**（火拼） |

结算同时发生：如 A 发育，B 和 C 都掠夺 A → A 扣 8，B 和 C 各加 5。

## 四、DM 职责
1. 宣布开局，公布玩家名单及初始金币
2. 主持发言 → 收行动 → 结算的流程
3. 每轮结算后公开战报摘要
4. 金币 ≤ 0 即宣布淘汰
