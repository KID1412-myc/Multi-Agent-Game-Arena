# 💣 《炸弹项圈》简化版 — 规则书

## 基础设定

| 项目 | 说明 |
|------|------|
| 人数 | 6 人（1 欺诈师 + 5 平民） |
| 局数 | 1 局 10 轮 |
| 初始积分 | 每人 5 分 |
| 存活奖励 | 每活过一轮 +1 分 |
| 秘密 | **DM 永不公布欺诈师的存在** |

## 核心装备：炸弹项圈

每人脖子上的项圈显示 **红、蓝、绿** 之一。**你看不到自己的颜色。** 每轮你只能看到另外 **2 个人** 的颜色（环形分配，确保每个人的颜色都有人看到）。欺诈师额外知道自己的颜色。

颜色每轮 100% 随机刷新。

## 每轮流程

| 阶段 | 内容 |
|------|------|
| 发言 | 全员公开发言。试探、结盟、质疑、说谎 |
| 猜色 | 每人秘密提交自己的颜色（红/蓝/绿） |
| 结算 | DM 公布谁爆炸了。猜对 +1 分存活，猜错爆炸淘汰。欺诈师免疫爆炸 |

## 审判（一局一次，隐藏机制）

审判的存在不提前告知玩家。当有人在公频质疑时，DM 才会宣布审判可用。

- 发起：花 **2 分**，公开指认一人
- 投票：全体存活玩家投票，**过半通过**
- 对了：欺诈师淘汰，指认者 **+4 分**（净赚 2）
- 错了：指认者爆炸淘汰
- 不通过：扣 2 分，无事发生

## 胜利条件

| 结局 | 条件 |
|------|------|
| 🟢 **平民胜利** | 欺诈师死亡（爆炸或审判） |
| 🟢 **平民共赢** | 连续 4 轮无人死亡 |
| 🟢 **标准结算** | 10 轮结束，存活者中积分最高获胜（平则共胜） |
| 😈 **欺诈师 A** | 存活到结束 + 积分最高 |
| 😈 **欺诈师 B** | 破坏和平 + 至少 2 个平民淘汰 + 自己存活 |
